# Технологічна практика  
## Інформаційна система обліку робочих змін працівників

## 📋 Про проєкт
**Duty** — настільний застосунок для обліку та формування графіків змін працівників водогосподарського підприємства.

## 👩‍💻 Виконала
**Студентка групи 44 — Богомаз Еліна**

## 🎯 Основні функції
- 📅 Перегляд та редагування графіків змін за місяць  
- 🔄 Автоматична генерація графіків на наступний місяць  
- 🔍 Пошук та фільтрація працівників за різними критеріями  
- 💾 Збереження даних у базі даних **SQLite**  
- 📊 Експорт графіків у формат **Excel**  
- 👥 Управління кадрами (додавання, редагування, видалення працівників)

## 🛠 Технологічний стек
- **Java 17+** — основна мова програмування  
- **JavaFX** — графічний інтерфейс користувача  
- **SQLite** — вбудована база даних  
- **Maven** — система збірки проєкту  
- **Apache POI** — робота з Excel-файлами
