# Технологічна практика — інформаційна система обліку робочих змін працівників
## Виконала: студентка 44 групи, Богомаз Еліна 

**Duty** — настільний застосунок для обліку та формування графіків змін працівників.
Реалізовано перегляд і редагування графіків за місяць, автоматичну генерацію наступного місяця, пошук і фільтрацію працівників, збереження даних у SQLite та експорт у формат Excel.

**Технології:** 
- Java 17+
- JavaFX
- QLite, Maven
- Apache POI
